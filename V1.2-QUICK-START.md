## рҹҗӣ Troubleshooting

### Problem: Nothing showing
**Cause:** Filters active from v1.1  
**Fix:** Click Г— on all tags to clear

### Problem: Select2 not working
**Cause:** jQuery/Select2 not loaded  
**Fix:** Hard refresh page (Ctrl+F5)

### Problem: Sliders not moving
**Cause:** Range not detected  
**Fix:** Check if items have prices/discounts

### Problem: Tags not appearing
**Cause:** No filters active  
**Fix:** Tags only show when filters applied

### Problem: Slow performance
**Cause:** Large wishlist (200+ items)  
**Fix:** Be patient, or filter more aggressively
